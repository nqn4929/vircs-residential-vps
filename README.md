## 为什么"真实家宽 IP"和"机房IP"差这么多？

Google、TikTok、Meta 这类平台对 IP 类型有非常精确的识别能力。机房 IP 的 ASN（自治系统号）归属是数据中心，哪怕商家说自己是住宅 IP，平台一眼就能看出来。结果就是：触发短信验证、账号权重压低、风控封号。
